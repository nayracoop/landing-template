# Landing básica multipropósito

Landing template en Vue.js para usar como base de proyectos simples.

## Instalación

Clonar repo y correr 
**`npm install`**

## Uso

 :zap: Ejecutar **`npm run dev`**

 :zap: Entrar a  http://localhost:8080/
